# Delícias da Beh — Confeitaria Artesanal

Bem-vindo(a) ao repositório do projeto **Delícias da Beh**, um site vitrine desenvolvido para uma confeitaria artesanal fictícia.
O objetivo é apresentar os produtos, contar a história da marca e permitir o contato com clientes interessados em encomendas personalizadas.

## Objetivo do Projeto
Criar um site moderno, atrativo e responsivo que represente a marca Delícias da Beh e facilite contato para encomendas.

## Páginas
- index.html — Página inicial
- doces.html — Galeria de produtos
- sobre.html — História e valores
- contato.html — Formulário de contato

## Tecnologias
- HTML5, CSS3, JavaScript
- Google Fonts (Poppins)
- Imagens ilustrativas (Unsplash)

## Como usar
1. Clone o repositório: `git clone https://github.com/SEU-USUARIO/delicias-da-beh.git`
2. Abra `index.html` no navegador para visualizar o site localmente.

## Observações
- Formulário de contato funciona via JavaScript (simulação).
- Para envio real de mensagens, integrar com backend ou serviço de e-mail.
